Polymer Gold Elements

The Polymer gold elements are specially designed for E-commerce use cases, like checkout flows.

Following is a list of different types of Polymer Gold elements:

Table:

Index Type Description
1) gold-cc-cvc-input The gold-cc-cvc is an input field used for the verification of a credit card.
2) gold-cc-input The gold-cc is an input field used for entering a credit card number.
3) gold-email-input The gold-email is an input field for entering an email address.
4) gold-phone-input The gold-phone is an input field for entering a phone number.
