Polymer Platinum Elements

The Polymer platinum elements are used to add features such as pushing notifications, offline usage, communicating with nearby bluetooth devices, etc. and provide features to convert a web page into a true web app.

See the elements of platinum in Polymer.js:

Table:

Index Type Description
1) Platinum Bluetooth It is used for interacting with nearby Bluetooth devices.
2) Platinum Push Messaging It is used to define the received push message.
